## The NOMAD Remote Tools Hub (NORTH)

NORTH allows you to run pre-configured applications directly on the NOMAD servers. With NORTH, you can access, analyze, and create data directly in your uploads using your favorite tools with no setup required.

With NORTH you will have everything you need for this tutoria, i.e., a **Linux terminal** and a **Python 3** installation with the required libraries.

Follow these steps to get started with NORTH:

1.  Begin by visiting the [NOMAD](https://nomad-lab.eu/nomad-lab/){:target="_blank"}. Once there, open NOMAD to reach the main interface.


2.  Navigate to the 'ANALYZE' section, which you'll find at the top of the page. Here, click on **NOMAD Remote Tools Hub** to access the remote tools available to you.

    <div style="text-align: center;">
        <img src="../images/1.png" alt="ANALYZE Section" width="800">
    </div>

3.  To use NORTH, you need to be logged in. Click on 'LOGIN / REGISTER' to proceed. If you are not yet registered, now is the right the time to join the NOMAD community.


    <div style="text-align: center;">
        <img src="../images/2.png" alt="Login page" width="800">
    </div>

4.  In the NORTH page you see several options. Let's start a new session by selecting 'Jupyter Notebook: The Classic Notebook Interface'. Click on **LAUNCH**, wait a few seconds and click **OPEN**.

    <div style="text-align: center;">
        <img src="../images/4.png" alt="Jupyter Notebook Interfac" width="800">
    </div>

    > **Note:** If a new page does not automatically appear, check to see if pop-ups are allowed in your browser, as this might prevent the notebook from opening. If you still see issues, try it with the Chrome browser.

    <div style="text-align: center;">
        <img src="../images/5.png" alt="Pop-up Allowance" width="800">
    </div>

5.  In the new page you see several tools, including a **Python 3 (ipykernel)** under Notebook and a **Terminal** under Other. So you have everything you need. for now, let's choose the **Python 3 (ipykernel)** to begin your session, which is the environment where you can run Python code.


    <div style="text-align: center;">
        <img src="../images/6.png" alt="Python 3 Kernel" width="800">
    </div>

6.  A Jupyter Notebook page opens, equipped with the libraries you need. This is where you can analyze data, visualize results, or interact with the NOMAD API.


    <div style="text-align: center;">
        <img src="../images/7.png" alt="Jupyter Notebook Page" width="800">
    </div>
